## 📌 About The Project

This website was built to provide a simple and visually appealing digital menu for customers.
It focuses on:

* Displaying menu items clearly
* Showing prices and images
* Making ordering quick and easy via phone

## 🚀 Deployment (GitHub Pages)

To deploy this site using GitHub Pages:

1. Go to your repository
2. Click **Settings**
3. Navigate to **Pages**
4. Under “Source”, select:

   * Branch: `main`
   * Folder: `/ (root)`
5. Click **Save**

Your site will be live in a few minutes 🎉
